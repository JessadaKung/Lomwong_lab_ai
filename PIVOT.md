# Pivot Worksheet

## 1. Domain

ชื่อธุรกิจ: Lomwong Cafe

ความเชื่อมโยงกับนักศึกษา: ธุรกิจคาเฟ่ที่ออกแบบจากบริบทนักศึกษาและคนทำงานใกล้มหาวิทยาลัยเทคโนโลยีราชมงคลอีสาน วิทยาเขตขอนแก่น

## 2. ลูกค้า + ปัญหา

ลูกค้าหลักคือใคร: นักศึกษา คนทำงาน และกลุ่มเพื่อนที่ต้องการเครื่องดื่มช่วงเย็นสำหรับอ่านหนังสือ คุยงาน หรือพักจากการเรียน

3 ปัญหาที่ระบบจะแก้:

1. ลูกค้าถามซ้ำเรื่องเวลาเปิด ที่ตั้ง เมนูแนะนำ และช่องทางสั่งล่วงหน้า
2. ลูกค้าเลือกเมนูไม่ถูก โดยเฉพาะคนที่ไม่ดื่มกาแฟหรือต้องการลดหวาน
3. เจ้าของร้านต้องตอบ DM เดิม ๆ หลายครั้ง ทำให้เสียเวลาช่วงเตรียมร้านและรับออเดอร์

## 3. Differentiation

ระบบนี้ต่างจาก MilkLab template เพราะปรับให้ตอบตามบริบท Lomwong Cafe โดยเฉพาะ: ร้านเปิดช่วงเย็นถึงดึกใกล้มหาวิทยาลัย, มี persona เป็นนักศึกษาและคนทำงาน, รองรับคำถามเรื่องเมนูกาแฟ/ชา/โกโก้/มัทฉะ, ระดับความหวาน, pre-order สำหรับรับหน้าร้าน และกิจกรรมกลุ่มเพื่อนหรือนักศึกษา
